# Voice Commands

Voice is codelord's signature feature. Hold a key to talk, release to execute. Whisper runs locally — nothing leaves your machine.

![Voice commands](assets/how-to/007-voice-commands.gif)

**How:** Hold the push-to-talk key. Say a command. Release.

**Starter commands:**

| Say this | Does this |
|----------|-----------|
| "open file" | Opens the file picker |
| "save" | Saves the current tab |
| "new tab" | Creates a new tab |
| "close tab" | Closes the current tab |
| "toggle terminal" | Shows/hides the terminal |
| "toggle explorer" | Shows/hides the file explorer |
| "toggle copilord" | Shows/hides the AI panel |
| "undo" / "redo" | Undo or redo |
| "play pause music" | Controls the music player |

> *Tip: You don't need to memorize all commands at once. Start with "open file", "save", and "toggle terminal" — the three you'll use most.*
