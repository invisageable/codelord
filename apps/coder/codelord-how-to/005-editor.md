# 005-editor

The text editor is where you spend most of your time. Tabs, saving, undo — the essentials.

![Editor basics](assets/how-to/004-editor-basics.gif)

**Key actions:**

| Action | Shortcut |
|--------|----------|
| New tab | `Cmd+T` |
| Close tab | `Cmd+W` |
| Next tab | `Cmd+}` or `Ctrl+Tab` |
| Previous tab | `Cmd+{` or `Ctrl+Shift+Tab` |
| Save | `Cmd+S` |
| Undo | `Cmd+Z` |
| Redo | `Cmd+Shift+Z` |

> *Tip: Unsaved tabs show a dot indicator. codelord will warn you before closing unsaved work.*
