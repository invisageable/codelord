# Panels

Three panels surround the editor. Toggle them on and off as you need them.

![Panels](assets/how-to/005-panels.gif)

**Layout:**

| Panel | Position | What it does |
|-------|----------|--------------|
| Explorer | Left | Browse project files |
| Copilord | Right | Local AI assistant |
| Terminal | Bottom | Run commands |

**How:** Use keyboard shortcuts or voice to toggle each panel. The editor resizes automatically.

> *Tip: You can say "toggle explorer" or "toggle terminal" with voice control too.*
