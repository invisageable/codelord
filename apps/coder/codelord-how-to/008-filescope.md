# 004-filescope.

The filescope lets you fuzzy-search for any file in your project. Type a few characters and it narrows down instantly.

> *NOTE: You need to have a project already opened.*

![filescope](assets/how-to/003-filescope.gif)

**How:** Press `Cmd+P` to open the filescope. Start typing a filename. Use arrow keys to select, `Enter` to open. `Esc` or toggle back `Cmd+P` to dismiss.

> *Tip: You don't need to type the full path — just a few characters from the filename is enough.*
