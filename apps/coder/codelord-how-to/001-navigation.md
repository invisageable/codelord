# 001-navigation.

On the top left corner, that's your navigation. It lets you navigate into all builtin simulations apps:

  - `welcome` — *The workstation screen.*
  - `editor` — *The code editor view, explorer, ai mentor.*
  - `playground` — *The zo real-time playground.*
  - `presenter` — *A quick presenter for conferences.*

1. You can reorder using drag and drop (can be buggy for now):

  ![navigation-navbar-reordering](assets/how-to/001-navigation-navbar-reordering.gif)

2. Use the navigation with your keyboard or you mouse:

  ![navigation-keybord-and-mouse](assets/how-to/001-navigation-keybord-and-mouse.gif)

**How:** Hold `Space` to open the XMB menu. Use arrow keys to move between categories (left/right) and items (up/down). Press `Enter` to select. Release `Space` to dismiss.
