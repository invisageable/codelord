# Themes & Settings

Make codelord yours. Switch themes, change font size, configure everything from the XMB settings menu.

![Themes and settings](assets/how-to/009-themes-and-settings.gif)

**How:** Open the XMB menu, navigate to **SETTiNGS**. Select **UI Theme** to switch between Dark and Light. Select **More** to open the full settings page.

**Settings categories:**

| Category | What you can change |
|----------|---------------------|
| Appearance | Theme, cursor animation, font size |
| Editor | Git blame, blame column width |
| App | Language, sounds, auto save, microphone |
| System | OS info, memory usage |

> *Tip: Settings are visual — no config files to edit unless you want to.*
