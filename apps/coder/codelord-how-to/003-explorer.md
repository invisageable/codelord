# 003-explorer.

The filescope lets you fuzzy-search for any file in your project. Type a few characters and it narrows down instantly.

> *NOTE: You need to have a project already opened.*

![explorer-empty-state](assets/how-to/003-explorer-empty-state.gif)

**How:** Click on OPEN  `Cmd+P` to open the filescope. Start typing a filename. Use arrow keys to select, `Enter` to open. `Esc` or toggle back `Cmd+P` to dismiss.

![explorer-tree-actions](assets/how-to/003-explorer-tree.actions.gif)

The toolbar at the top shows the items count and quick actions:

  - Refresh — *reload the file tree*
  - Collapse all — *fold every directory*
  - Toggle hidden files — *show/hide dotfiles*

![explorer-tree-actions](assets/how-to/003-explorer-tree-actions.gif)

Right-click any file or folder for the context menu:

| Action                             | What it does                          |
| :--------------------------------- | :------------------------------------ |
| New file                           | Create a file in this directory       |
| New folder                         | Create a folder in this directory     |
| Add to workspace                   | Add a root folder to the workspace    |
| Remove from workspace              | Remove a root folder                  |
| Cut                                | Cut for move                          |
| Copy                               | Copy for paste                        |
| Paste                              | Paste cut/copied item                 |
| Copy path                          | Copy the absolute path                |
| Copy relative path                 | Copy the path relative to the project |
| Reveal in Finder                   | Open the parent folder in Finder      |
| Rename                             | Rename the file or folder             |
| Delete                             | Delete the file or folder             |

![explorer-right-click](assets/how-to/003-explorer-right-click.gif)
